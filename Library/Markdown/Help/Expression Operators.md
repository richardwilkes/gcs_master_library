### Expression Operators

| Operator | Name                     |
|:--------:| ------------------------ |
| (        | Open Parenthesis         |
| )        | Close Parenthesis        |
| +        | Add                      |
| -        | Subtract                 |
| ^        | Power                    |
| *        | Multiply                 |
| /        | Divide                   |
| ==       | Equal To                 |
| !=       | Not Equal To             |
| <=       | Less Than or Equal To    |
| <        | Less Than                |
| >=       | Greater Than or Equal To |
| >        | Greater Than             |
| &&       | Logical And              |
| \|\|     | Logical Or               |
| !        | Logical Not              |
| %        | Modulo                   |

See also: [Expression Functions](./Expression%20Functions.md)
