# Scripting: Entity

> [Return to the Scripting Guide.](../Scripting%20Guide)

This object represents the current character sheet data.

| Property               | Type     | Value                                                                                                                                                                                                     | Availability |
|------------------------|----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| exists                 | boolean  | True if this script is being run on a sheet. If this is false, all other properties will be undefined and only the `randomHeightInInches` and `randomWeightInPounds` methods will be defined (see below). | GCS v5.36+   |
| playerName             | string   | The player’s name.                                                                                                                                                                                        | GCS v5.36+   |
| name                   | string   | The character’s name.                                                                                                                                                                                     | GCS v5.36+   |
| title                  | string   | The character’s title.                                                                                                                                                                                    | GCS v5.36+   |
| organization           | string   | The character’s organization.                                                                                                                                                                             | GCS v5.36+   |
| religion               | string   | The character’s religion.                                                                                                                                                                                 | GCS v5.36+   |
| techLevel              | string   | The character’s Tech Level.                                                                                                                                                                               | GCS v5.36+   |
| gender                 | string   | The character’s gender.                                                                                                                                                                                   | GCS v5.36+   |
| age                    | string   | The character’s age.                                                                                                                                                                                      | GCS v5.36+   |
| birthday               | string   | The character’s birthday.                                                                                                                                                                                 | GCS v5.36+   |
| eyes                   | string   | The character’s eyes.                                                                                                                                                                                     | GCS v5.36+   |
| hair                   | string   | The character’s hair.                                                                                                                                                                                     | GCS v5.36+   |
| skin                   | string   | The character’s skin.                                                                                                                                                                                     | GCS v5.36+   |
| handedness             | string   | The character’s primary hand.                                                                                                                                                                             | GCS v5.36+   |
| heightInInches         | number   | The character’s height, in inches.                                                                                                                                                                        | GCS v5.36+   |
| weightInPounds         | number   | The character’s weight, in pounds.                                                                                                                                                                        | GCS v5.36+   |
| displayHeightUnits     | string   | The height units that should be used for display.                                                                                                                                                         | GCS v5.36+   |
| displayWeightUnits     | string   | The weight units that should be used for display.                                                                                                                                                         | GCS v5.36+   |
| sizeModifier           | number   | The character’s Size Modifier.                                                                                                                                                                            | GCS v5.36+   |
| liftingStrength        | number   | The character’s ST for lifting purposes.                                                                                                                                                                  | GCS v5.36+   |
| strikingStrength       | number   | The character’s ST for striking purposes.                                                                                                                                                                 | GCS v5.36+   |
| throwingStrength       | number   | The character’s ST for throwing purposes.                                                                                                                                                                 | GCS v5.36+   |
| extraDiceFromModifiers | boolean  | True if the Modifying Dice + Adds option from B269 is in use.                                                                                                                                             | GCS v5.36+   |
| attributes             | object[] | The list of [Attributes](Attribute) for this sheet.                                                                                                                                                       | GCS v5.36+   |
| encumbrance            | object   | The [Encumbrance](Encumbrance) data for this sheet.                                                                                                                                                       | GCS v5.36+   |
| equipment              | object[] | The top-level list of carried [Equipment](Equipment) for this sheet with a quantity greater than 0.                                                                                                       | GCS v5.36+   |
| skills                 | object[] | The top-level list of [Skills](Skill) for this sheet.                                                                                                                                                     | GCS v5.36+   |
| spells                 | object[] | The top-level list of [Spells](Spell) for this sheet.                                                                                                                                                     | GCS v5.36+   |
| traits                 | object[] | The top-level list of enabled [Traits](Trait) for this sheet.                                                                                                                                             | GCS v5.36+   |
| weapons                | object[] | The list of enabled [Weapons](Weapon) for this sheet.                                                                                                                                                     | GCS v5.41+   |
| points                 | object   | The [Points](Points) breakdown data for this sheet.                                                                                                                                                       | GCS v5.43+   |

| Method               | Arguments                                                                               | Result Type | Result                                                                                                                                                                                                                                                                                                                                                                                                      | Availability           |
|----------------------|-----------------------------------------------------------------------------------------|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------|
| attribute            | idOrName: string                                                                        | object      | Returns the [Attribute](Attribute) that matches the given ID or name. IDs are checked first, then names. Names are compared case-insensitively.                                                                                                                                                                                                                                                             | GCS v5.36+             |
| currentEncumbrance   | forSkills: boolean, returnMoveFactor: boolean                                           | number      | Returns the current encumbrance modifier (a value from 0 to 4) or the move multiplication factor if **returnMoveFactor** is true. If **forSkills** is true, the modifier is adjusted to account for equipment that has been marked as not having its weight counting against skills.                                                                                                                        | GCS v5.36+             |
| findEquipment        | name: string, tag: string                                                               | object[]    | Returns the carried [Equipment](Equipment) that match the given **name** and has the given **tag,** case-insensitively. When matching, an empty or undefined value passed in will match everything. Note that only equipment with a quantity greater than 0 is considered and all parents of that equipment must also have a quantity greater than 0.                                                       | GCS v5.36+             |
| findSkills           | name: string, specialization: string, tag: string                                       | object[]    | Returns the [Skills](Skill) that match the given **name**, the given **specialization,** and have the given **tag,** case-insensitively. When matching, an empty or undefined value passed in will match everything.                                                                                                                                                                                        | GCS v5.36+             |
| findSpells           | name: string, tag: string                                                               | object[]    | Returns the [Spells](Spell) that match the given **name** and have the given **tag,** case-insensitively. When matching, an empty or undefined value passed in will match everything.                                                                                                                                                                                                                       | GCS v5.36+             |
| findTraits           | name: string, tag: string                                                               | object[]    | Returns the enabled [Traits](Trait) that match the given **name** and have the given **tag,** case-insensitively. When matching, an empty or undefined value passed in will match everything.                                                                                                                                                                                                               | GCS v5.36+             |
| findWeapons          | melee: boolean, name: string, usage: string                                             | object[]    | Returns the enabled [Weapons](Weapon) that match the given **name** and **usage**, case-insensitively. When matching, an empty or undefined value passed in will match everything.                                                                                                                                                                                                                          | GCS v5.41+             |
| hasTrait             | name: string                                                                            | boolean     | Returns true if the named [Trait](Trait) is present and enabled.                                                                                                                                                                                                                                                                                                                                            | GCS v5.36+             |
| skillLevel           | name: string, specialization: string, relative: boolean, optionalSpecialization: string | number      | Returns the level of the [Skill](Skill) with the given **name** and **specialization.** If **relative** is true, returns the relative level instead. Note that the fourth parameter, **optionalSpecialization** was not introduced until GCS v5.43                                                                                                                                                          | GCS v5.36+, GCS v5.43+ |
| traitLevel           | name: string                                                                            | number      | Returns the combined level of the enabled trait(s) with the given ’name’, or -1 if none are found.                                                                                                                                                                                                                                                                                                          | GCS v5.36+             |
| randomHeightInInches | strength: number                                                                        | number      | Returns a height in inches based on the given **strength** using the chart from B18.                                                                                                                                                                                                                                                                                                                        | GCS v5.36+             |
| randomWeightInPounds | strength: number, shift: number                                                         | number      | Returns a weight in pounds based on the given **strength** using the chart from B18. Adjusts appropriately for the traits _Skinny, Overweight, Fat,_ and _Very Fat,_ if present on the sheet. **shift** causes a shift towards a lighter value if negative and a heavier value if positive, similar to having one of the traits _Skinny, Overweight, Fat,_ and _Very Fat_ applied, but is additive to them. | GCS v5.36+             |
| swingFor             | value: number                                                                           | string      | Returns the damage dice that would result if the value was used as your strength when calculating swing damage.                                                                                                                                                                                                                                                                                             | GCS v5.43+             |
| thrustFor            | value: number                                                                           | string      | Returns the damage dice that would result if the value was used as your strength when calculating thrust damage.                                                                                                                                                                                                                                                                                            | GCS v5.43+             |
