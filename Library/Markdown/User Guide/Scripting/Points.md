# Scripting: Points

> [Return to the Scripting Guide.](../Scripting%20Guide)

This object holds a points breakdown for an [Entity](Entity).

| Property      | Type   | Value                                                                 | Availability |
|---------------|--------|-----------------------------------------------------------------------|--------------|
| total         | number | The total points. Note that this does not include any unspent points. | GCS 5.43+    |
| unspent       | number | The unspent points.                                                   | GCS 5.43+    |
| ancestry      | number | The points spent on an Ancestry.                                      | GCS 5.43+    |
| attributes    | number | The points spent on attributes.                                       | GCS 5.43+    |
| advantages    | number | The points spent on advantages.                                       | GCS 5.43+    |
| disadvantages | number | The points spent on disadvantages.                                    | GCS 5.43+    |
| quirks        | number | The points spent on quirks.                                           | GCS 5.43+    |
| skills        | number | The points spent on skills.                                           | GCS 5.43+    |
| spells        | number | The points spent on spells.                                           | GCS 5.43+    |
